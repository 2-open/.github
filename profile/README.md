# Hey, we’re 2open 👋

We’re building **2open**, an open-source email campaign manager for founders, indie hackers, agencies, and small teams who want to send campaigns from their own SMTP without bloated marketing software.

## What we do

2open helps you import leads, create campaigns, send emails, and track opens/clicks — all from a simple self-hostable dashboard.

## Why 2open exists

Most email tools are expensive, closed, or overloaded with features small teams don’t need.

2open is built to be simple, transparent, and open-source first.

## Who it’s for

- Indie hackers
- SaaS founders
- Agencies
- Developers
- Small teams

## What you can do with 2open

- Import leads from CSV
- Send campaigns through your own SMTP
- Track opens and clicks
- Manage workspaces and campaigns
- Self-host with Docker
- Use the hosted version at **2open.email**

## Open source first

The core product is open source so anyone can inspect it, self-host it, customize it, and build on top of it.

Hosted version: **https://2open.email**
